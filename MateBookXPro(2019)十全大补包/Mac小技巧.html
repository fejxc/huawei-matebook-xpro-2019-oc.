MAC 小技巧

一、关闭update小红点

defaults write com.apple.systempreferences AttentionPrefBundleIDs 0

二、自动隐藏程序坞零延时的弹出

开启
defaults write com.apple.Dock autohide-delay -float 0 && killall Dock
关闭（恢复原来默认的延迟速度）
defaults delete com.apple.Dock autohide-delay && killall Dock

三、不挂载部分分区(类似于隐藏分区)

编辑/etc/fstab文件-有Linux基础可以直接vi，无基础着建议编辑文本后拷贝进去
#NTFS（UUID使用Clover Configurator看到的卷UUID，下面两行任意一个均可）
UUID=你想要隐藏的分区卷宗的UUID none ntfs ro,noowners,noauto
LABEL=你想要隐藏的分区名称 none ntfs ro,noowners,noauto
#FAT
LABEL=你想要隐藏的分区名称none msdos ro,noowners,noauto

四、调整 Launchpad 图标

调整每列显示图标数量，这里以 7 为例：
defaults write com.apple.dock springboard-rows -int 7
调整每行显示图标的数量，这里以6 为例：
defaults write com.apple.dock springboard-columns -int 6
上面两行代码最后的数字可以自行修改。
修改完了之后还需要重置一下 Launchpad，代码如下：
defaults write com.apple.dock ResetLaunchPad -bool TRUE;killall Dock

五、MAC、linux、windows时间同步

mac和linux认为bios时间为utc时间，windows认为是本地时间
统一设置为BIOS时间为UTC时间，Windows设置如下：
开 始->运行->CMD，打开命令行程序，在命令行中输入下面命令并回车
Reg add HKLM\SYSTEM\CurrentControlSet\Control\TimeZoneInformation /v RealTimeIsUniversal /t REG_DWORD /d 1

六、无鼠标重建缓存

1、使用"command+空格键"弹出聚焦搜索
2、输入terminal,下方搜索框会弹出终端，回车即可打开"终端"
4、输入 sudo kextcache -i /
5、等待完成
6、输入 sudo reboot ;重新启动机器

七、命令行重建缓存

sudo kextcache -i /
