Matebook X pro 2019 设置 dvmt=64M

提醒: 如非必须外接4K,不必进行该操作

提醒: 该说明仅仅针对Matebook X pro 2019

提醒: 该操作一定要在自己机器上进行,使用他人修改后的结果文件,有可能导致机器变砖

该设置在windows下进行,可以通过Bios设置的"恢复出厂设置"复原

如果文档看不懂,部分步骤不知道如何操作,建议停止,不要折腾,这个没有容错的机会.也不要抱怨我写的简单

一、解压工具

windows中解压工具 H20UVE_100.00.9.2

二、导出现有BIOIS设置

用管理员权限运行cmd.exe,并通跳转到H2OUVE目录,输入命令

  H2OUVE.exe -gv vars-bios-setup.txt

三、备份当前设置

copy vars-bios-setup.txt vars-bios-setup-change.txt

四、编辑 vars-bios-setup-change.txt

使用文本编辑器打开文件vars-bios-setup-change.txt
搜索 72C5E28C-7783-43A1-8767-FAD73FCCAFA4,一共有两个,一个是SaSetup,一个是Custom; 修改Setup段中数据.

找到00000100:
后面第8个数字即0X107位:原值为01，修改为02，
后面第9个数字即0X108位:原值为02，修改为03，

在管理员权限cmd.exe输入命令
